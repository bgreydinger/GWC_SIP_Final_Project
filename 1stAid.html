First Aid


*Disclaimer: This article is only to be used as a guide for basic First Aid. If you are concerned that there is a more serious medical issue, please call 911 and heed the direction of a licensed physician.

Basic First Aid for Bleeding:

Cover the wound with gauze or cloth and apply direct pressure. Add more layers till the blood flow stops.

Basic First Aid for Burns:

1. Flush the burned area with cool running water

2. Apply a light gauze bandage

3. Take pain relief medication if necessary

4. Do not break formed blisters

Basic First Aid for Sprains:

1. Immobilize the limb

2. Apply a cold pack

3. Elevate it

4. See Doctor for further diagnosis and treatment

Basic First Aid for Bee Stings:

1. Take out the stinger

2. If the person is allergic to bee stings, administer an EpiPen

3. Apply a cold pack

4. Use an antihistamine like Benadryl (diphenhydramine) to reduce swelling and itching.

5. Take pain relief medication
